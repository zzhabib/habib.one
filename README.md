# habib.one

My portfolio site.

Built with React, TypeScript, Tailwind CSS, and Vite. Features an aurora borealis WebGL background, scroll-reveal animations, and English/Japanese language toggle.

## Stack

- React 19 + TypeScript
- Tailwind CSS v4
- Three.js
- Vite

## Credits

Aurora shader by [nimitz](https://www.shadertoy.com/view/XtGGRt)

## Dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
